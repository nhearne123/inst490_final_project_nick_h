# Fruit Finder + Favorites (INST377 Final Project)

## Project Overview

### Title of Project
Fruit Finder + Favorites

### Description of Project
Fruit Finder + Favorites is a simple full-stack web application that lets users:
- Browse fruit data from the Fruityvice API (served through this project’s backend)
- Search for a fruit by name and view nutrition facts
- View a “Low Sugar” list (top 10 fruits with the lowest sugar)
- Save favorite fruits to a Supabase database and manage them (view + delete)

The frontend is a multi-page website located in the `/public` folder, and it accesses all data using **Fetch API** calls to this project’s REST API endpoints.

### Target Browsers
This project is designed to work on modern desktop and mobile browsers:
- **Desktop:** Chrome (latest), Firefox (latest), Safari (latest), Microsoft Edge (latest)
- **Mobile:** iOS Safari (latest), Android Chrome (latest)
